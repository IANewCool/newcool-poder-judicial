# NewCool Poder Judicial - Tribunales Chile

Portal educativo sobre el Poder Judicial de Chile.

## Demo

[https://poder-judicial.newcool.io](https://poder-judicial.newcool.io)

## Caracteristicas

- Buscador de causas judiciales
- Estructura del Poder Judicial
- Guia procesal
- Directorio de tribunales
- Consulta de juzgados

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
